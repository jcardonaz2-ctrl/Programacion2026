# Programacion2026

1.Que es un git
Git es un sistema de control de versiones distribuido que permite Guardar,organizar y controlar
los cambios que se hacen archivos o proyectos.

2.Cuales son los comandos mas importantes y para que sirven
Git es un sistema de control de versiones distribuido que rastrea cambios en el código fuente durante el desarrollo de software, facilitando la colaboración y el historial de cambios. Los comandos esenciales incluyen git init, git clone, git add, git commit, git push, git pull, git branch y git merge. 

3.Cual es la diferencia entre Git y Github
Git es un sistema de control de versiones local que rastrea cambios en el código en tu computadora, mientras que GitHub es una plataforma en la nube que aloja esos repositorios de Git para compartir y colaborar. En resumen: Git es la herramienta y GitHub es el servicio en la nube para trabajar con ella. 

4.Comandos
1.git status: Verifica qué archivos han sido modificados.
2.git add . o git add <archivo>: Prepara los cambios para el commit.
3.git commit -m "Mensaje claro": Confirma los cambios en tu repositorio local.
4.git push origin <rama: Sube los cambios confirmados al repositorio remoto. 

5. Que es una branch
Una branch (rama) en programación (especialmente en Git) es una línea de desarrollo independiente que permite trabajar en nuevas funcionalidades o corregir errores sin afectar el código principal o "master". Es un concepto fundamental para la colaboración, permitiendo aislar cambios antes de integrarlos.
